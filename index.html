Save The Box

Save The Box is a simple web-based game where the player's goal is to move a box across the screen and avoid obstacles. The box moves automatically, and the player must use arrow keys to control its direction. The game becomes progressively challenging as the box moves faster, and the obstacles become more frequent. The project was built using HTML, CSS, and JavaScript. I implemented the game logic, including collision detection, score tracking, and game over handling. Additionally, I added some CSS animations to make the gameplay more engaging. Give it a try and see how high you can score in Save The Box!
